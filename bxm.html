BricksXMortar

Goal

In this project we were asked to design and create an interactive online experience that would act as the primary branding tool for a fictional local fashion studio called BRICKSxMORTAR. It was desired to have a responsive and mobile-friendly website layout that would seamlessly work across different devices. As a technology the use of JavaScript and jQuery frameworks with HTML & CSS based layouts was highly encouraged.

Process:

BRICKSxMORTAR was defined as a studio that creates one-of-a-kind custom clothes for clients. After researching about custom clothing and different body shapes I realized women especially younger women have more problems with their bodies which may lead them to feel uncomfortable and insecure.

Result

I decided to create a welcoming, informative and a feminine website for BRICKSxMORTAR. I believe in the importance of knowing yourself and your qualities and accepting them as it is. I mapped the colours of the website to the logo we were all given, but since I also wanted to create a fun and interactive environment for the visitors, I used colourful graphics anywhere I could.

Skills

Since I am neither a graphic designer nor a website developer, this was a challenging assignment. Due to tight deadlines, I had to be constantly alert and quick in learning different languages and creating layouts. At the end of the project I gained the capability to code with JavaScript and jQuery frameworks with HTML & CSS based layouts.
